> 🌐 **GAZELOOM** 是一个基于深度学习与计算机视觉的驾驶员凝视估计系统。  
> 它融合眼动追踪、头部姿态分析与注意力建模，实现对驾驶员关注区域的实时识别与预测。  
>  
> 🚘 **应用场景**：疲劳检测 · 驾驶行为分析 · 智能座舱 · 人机共驾交互  

---

### 🧠 核心功能 Highlights
- 🔹 **实时凝视估计**：精准识别驾驶员视线方向与关注区域  
- 🔹 **多模态融合**：人脸、眼睛、姿态与环境特征联合建模  
- 🔹 **智能风险预警**：疲劳驾驶检测与异常行为分析  
- 🔹 **数据平台接口**：可视化凝视轨迹与行为统计
